## Why

O projeto é de um desenvolvedor brasileiro e o público-alvo inclui a comunidade brasileira de Flutter. Ter uma versão em português do README facilita o acesso e compreensão para desenvolvedores que preferem documentação em PT-BR.

## What Changes

- Criar arquivo `README.pt-br.md` na raiz do projeto com todo o conteúdo do README.md traduzido para português brasileiro
- Adicionar link no README.md (inglês) apontando para a versão PT-BR

## Capabilities

### New Capabilities
- `readme-pt-br`: Versão em português brasileiro do README com todas as seções traduzidas (arquitetura, features, setup, dependências, API)

### Modified Capabilities
- `readme-documentation`: Adicionar link para versão PT-BR no README principal em inglês

## Impact

- Novo arquivo: `README.pt-br.md` (raiz do projeto)
- Arquivo modificado: `README.md` (adicionar link para versão PT-BR)
- Nenhuma alteração de código ou dependências
