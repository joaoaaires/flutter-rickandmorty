## Why

O README atual é extremamente básico — contém apenas o nome do projeto, um link para a API e uma menção ao GetX. Não documenta a arquitetura, estrutura de pastas, dependências, como rodar o projeto, nem as funcionalidades implementadas. Isso dificulta a contribuição de novos desenvolvedores e a manutenção do projeto.

## What Changes

- Reescrever o README.md com documentação completa do projeto
- Adicionar seção de arquitetura (Clean Architecture + GetX)
- Adicionar diagrama de estrutura de pastas
- Documentar todas as features (Character, Episode, Location)
- Listar dependências com suas finalidades
- Adicionar instruções de setup e execução (Android, iOS, Web)
- Manter o GIF demonstrativo existente
- Adicionar seção de API utilizada com endpoints consumidos
